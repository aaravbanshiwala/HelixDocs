## What is HelixCMS?

**HelixCMS** is the next generation of **Self-Hosted Minecraft Server CMS**. It is a modern, fast, and easy-to-use website CMS that allows you to convert your \*\*audience \*\*on your Minecraft server. It is built with the latest technologies and is designed to be easy to use and customize.
